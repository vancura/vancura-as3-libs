pad

public static function pad(value: Number,  
beforePoint: uint,  
afterPoint: uint = ):String

Pads the value with the set number of digits before and after the point.  If the number of digits in the integer of value is less than beforePoint, the remaining digits are filled with zeros.  If the number of digits in the decimal of value is less than afterPoint, the remaning digits are filled with zeros.

Parameters

valuethe number to pad
beforePointthe number of digits to pad before the point
afterPointthe number of digits to pad after the point
