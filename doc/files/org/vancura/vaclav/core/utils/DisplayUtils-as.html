getParents

public static function getParents(obj: DisplayObject,  
includeSelf: Boolean = true,
stopAt: DisplayObject = null):Array

Get the parents of a display object as an array.  Original article: http://jacksondunstan.com/articles/519

Parameters

objObject whose parents should be retrieved
includeSelfIf obj should be included in the returned array
stopAtDisplay object to stop getting parents at.  Passing null indicates that all parents should be included.

Returns

An array of the parents of the given display object.  This includes all parents unless stopAt is non-null.  If stopAt is non-null, it and its parents will not be included in the returned array.
