embed2BD

public static function embed2BD(source: *):BitmapData

Get BitmapData from the source.  If the source is Bitmap, it’s converted, if it is already BitmapData, the reference is returned.  Serves as a quick filter for Bitmap and BitmapData.  Throws a TypeError if the source is not Bitmap nor BitmapData.
